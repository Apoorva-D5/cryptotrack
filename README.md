A full-stack crypto tracking platform that allows users to monitor live prices, manage their portfolio, track profit/loss, create a personalized watchlist, and view interactive charts.
 Features
1. Authentication
- JWT-based login & register  
- Protected routes
2.Live Crypto Prices
- Top 250 coins from CoinGecko
- Search by name  
- Sort & filter  
- Pagination  
- Sparklines (7-day chart)
3. Personal Watchlist
- Add/remove any coin  
- Live price updates  
- Instant UI sync
4. Portfolio Manager
- Add crypto holdings  
- Track total investment  
- Auto-calculated current value  
- Real-time profit/loss  
- Delete entries  
- Summary cards

##  Tech Stack
Frontend:
- React.js  
- TailwindCSS  
- Axios
- Recharts  

Backend:
- Node.js  
- Express  
- JWT authentication  
- MongoDB / Mongoose  

API:
- CoinGecko REST API (live crypto data)
